# Next.js

## tutorial

[Next.js Introduction](https://nextjs.org/learn/basics/create-nextjs-app/setup)

```sh
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"
```
